电子科技大学本科生自动健康打卡。

## 💡 项目介绍

本项目能够在每日的 00:30 为您自动上报体温，并在 14:30 进行复查。

如果您没有禁用 GitHub 的邮件通知，那么万一上报失败，您将收到一封邮件提醒。

## 📚 部署教程
